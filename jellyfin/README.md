# Home Assistant Add-on: Jellyfin

Jellyfin is a free and open source media server. [web:284]

This add-on uses the official `jellyfin/jellyfin:latest` container image and stores its configuration under `/share/jellyfin`, while your libraries are exposed from `/media`. [web:284][web:292][page:0]

## First setup

1. Install and start the add-on.
2. Open the Web UI on port 8096.
3. Complete the initial setup wizard.
4. Add libraries from `/media`. [web:284][page:0]
