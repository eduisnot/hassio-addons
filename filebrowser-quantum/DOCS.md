# FileBrowser Quantum

This add-on uses FileBrowser Quantum with configuration stored in `/share/filebrowser`. The Docker documentation states the default config and database are stored under the data directory, and also notes that config and database paths can be set manually. [page:0]

## Exposed paths

- `/ha_config`
- `/ssl`
- `/addons`
- `/backup`
- `/share`
- `/media`
- `/addon_configs` [page:2]

The web UI is exposed on host port 8383. [page:2]
